Store product images in this folder and use relative paths in admin.

Examples:
- assets/products/sofa-main.jpg
- assets/products/sofa-side.jpg
- assets/products/wardrobe-1.webp

How to use:
1. Copy your image files into `assets/products/`
2. Open `admin.html`
3. In `Main Image URL`, enter a relative path like `assets/products/sofa-main.jpg`
4. In `Additional Gallery Image URLs`, enter one extra path per line

These files will be served by Firebase Hosting when you deploy the site.
